# indexnow-notifier

A simple tool to notify search engines about new or updated content on your website using the IndexNow protocol.

## Installation

You can install indexnow-notifier using npm:

```bash
npm install -g indexnow-notifier
```

## Usage

To notify search engines about new or updated content, run the following command:

```bash
indexnow --site https://example.com --host example.com --key your-api-key --framework your-framework
```

### Options

- `--site`: The URL of your website (e.g., https://example.com).
- `--host`: The host name of your website (e.g., example.com).
- `--key`: Your API key for authentication with the search engines.
- `--framework`: The framework you are using (e.g., nextjs-app, nextjs-pages etc.).

## Supported Frameworks

- Next.js (app and pages)
